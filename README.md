# go-proxies

### Pre-requisites
- [Gorm](https://github.com/go-gorm/gorm)
- [Proxy](https://medium.com/swlh/proxy-server-in-golang-43e2365d9cbc)


## Installation

A little intro about the installation. 
```
$ https://github.com/mauryasaurav/go-proxies.git
$ cd go-proxies
$ go mod tidy
$ go run cmd/main.go
```
